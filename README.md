# Customer Behaviour EDA

## Project Overview
This project analyzes customer purchasing behaviour using the Superstore Sales Dataset using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset Information
- Total Records: 9,994
- Total Columns: 21
- No Missing Values

## Analysis Performed
- Data Inspection
- Data Cleaning Validation
- Descriptive Statistics
- Missing Value Analysis
- Sales Analysis
- Customer Behaviour Analysis
- Profit Analysis

## Key Insights
- Technology category generated the highest sales.
- West region contributed the highest sales.
- Customer purchasing behaviour varies across segments.
- Profitability differs across categories and products.

- Customer_Behaviour_EDA.ipynb
- Sample - Superstore.csv
